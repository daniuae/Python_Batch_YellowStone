# Java Fundamentals

## JDK, JRE, JVM Architecture

### JVM (Java Virtual Machine)

The JVM is the engine that executes Java programs.

**Responsibilities:**

* Loads bytecode
* Verifies bytecode
* Executes code
* Manages memory
* Performs Garbage Collection

### JVM Architecture

```text
        Java Source Code
               |
            javac
               |
          Bytecode (.class)
               |
         +-------------+
         |     JVM     |
         +-------------+
               |
      Machine Specific Code
               |
            OS/Hardware
```

---

### JRE (Java Runtime Environment)

**JRE = JVM + Libraries**

Provides:

* JVM
* Core Java Libraries
* Runtime Environment

Used only to run Java programs.

```text
JRE
 ├── JVM
 └── Java Libraries
```

---

### JDK (Java Development Kit)

**JDK = JRE + Development Tools**

Contains:

* javac
* java
* javadoc
* debugger
* JVM
* Libraries

```text
JDK
 ├── JRE
 │    ├── JVM
 │    └── Libraries
 └── Development Tools
```

## Java Compilation Process

### Step 1: Write Source Code

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello Java");
    }
}
```

Save as:

```text
HelloWorld.java
```

### Step 2: Compile

```bash
javac HelloWorld.java
```

Output:

```text
HelloWorld.class
```

### Step 3: Run

```bash
java HelloWorld
```

Output:

```text
Hello Java
```

---

## Bytecode

Bytecode is the intermediate code generated by the Java compiler.

```text
HelloWorld.java
      |
    javac
      |
HelloWorld.class
      |
      JVM
      |
Machine Code
```

**Benefits:**

* Platform Independent
* Secure
* Portable

## Java Data Types

### Primitive Data Types

| Type    | Size          | Example |
| ------- | ------------- | ------- |
| byte    | 1 byte        | 10      |
| short   | 2 bytes       | 100     |
| int     | 4 bytes       | 1000    |
| long    | 8 bytes       | 10000L  |
| float   | 4 bytes       | 10.5f   |
| double  | 8 bytes       | 10.5    |
| char    | 2 bytes       | 'A'     |
| boolean | JVM dependent | true    |

### Example

```java
public class DataTypes {
    public static void main(String[] args) {

        int age = 25;
        double salary = 45000.50;
        char grade = 'A';
        boolean status = true;

        System.out.println(age);
        System.out.println(salary);
        System.out.println(grade);
        System.out.println(status);
    }
}
```

## JVM Memory Areas

### Stack Memory

Stores:

* Local Variables
* Method Calls
* References

Example:

```java
int x = 10;
```

Stored in Stack Memory.

---

### Heap Memory

Stores:

* Objects
* Arrays

Example:

```java
Student s = new Student();
```

The object is stored in Heap Memory, while the reference variable is stored in Stack Memory.

### Stack and Heap Diagram

```text
Stack Memory

main()
 |
 |-- x = 10
 |
 |-- s ------+
             |
             v

Heap Memory
+----------------+
| Student Object |
| age = 25       |
+----------------+
```

---

## Garbage Collection

Unused objects are automatically removed by the Garbage Collector.

```java
Student s = new Student();

s = null;
```

The object becomes eligible for Garbage Collection.

## Java Coding Standards

### Class Names

Use **PascalCase**

```java
EmployeeDetails
CustomerAccount
```

### Method Names

Use **camelCase**

```java
calculateSalary()
displayEmployee()
```

### Variable Names

Use **camelCase**

```java
employeeName
employeeSalary
```

### Constants

Use **UPPER_CASE**

```java
static final double PI = 3.14159;
```

### Proper Indentation

Good:

```java
if (age > 18) {
    System.out.println("Adult");
}
```

Bad:

```java
if(age>18){System.out.println("Adult");}
```

# Complete Example Program

```java
import java.util.Scanner;

public class Employee {

    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);

        System.out.print("Enter Name: ");
        String name = sc.nextLine();

        System.out.print("Enter Salary: ");
        double salary = sc.nextDouble();

        double bonus = salary * 0.10;

        System.out.println("Employee Name : " + name);
        System.out.println("Salary        : " + salary);
        System.out.println("Bonus         : " + bonus);

        sc.close();
    }
}
```
